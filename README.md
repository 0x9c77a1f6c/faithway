### Installation
1. Clone the repository:
  ```bash
   git clone https://github.com/0x9c77a1f6c/faithway.git
   cd faithway
  ```

2. Open the project in Xcode:
```bash
  open faithway.xcodeproj
```

3. Build and run the app:
- Select a simulator (e.g., iPhone 14 Pro)
- Press Cmd + R to run
